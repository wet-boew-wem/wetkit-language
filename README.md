WxT Languages
=============
Integrates multilingual support for [Drupal WxT][drupalwxt].

Key Features
------------

* Bilingual tokens
* Integration with [i18n][i18n] and [Entity Translation][entity_trans]
* Language Switcher

Important
---------

* Integrates the work done in: [http://drupal.org/sandbox/daspeter/1444130][daspeter]
